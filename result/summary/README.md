# summary 目录说明

本目录保存 Lab8 优化结果的总览文档和最终方案说明。这里的文件用于快速了解各版本效果、最终选择原因和完整迭代过程。

## 文件说明

- `version_metrics_summary.md`：各版本的综合指标汇总表，包含 SSE、latency、power、历史 score 以及每个版本的关键说明。
- `final_selected_scheme.md`：最终选定方案说明，记录最终架构、主要参数、优化点和采用该方案的原因。
- `optimization_history.md`：优化迭代流程记录，包含每轮实验的执行规范、统计口径、停止条件和历史优化计划。
